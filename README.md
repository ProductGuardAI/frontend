# ProductGuard Frontend

Responsive React/Vite reviewer application on port `5173`. Run `npm run dev -w frontend`, then open `/dashboard` or `/demo`. The application includes dashboard metrics, filtering, case creation, secure file pre-validation, processing progress, complete review tabs, editable/verified attributes, findings actions, decisions, activity, and responsive layouts. Configure `VITE_API_URL`; private provider keys never belong here.
